2 操作：
	将需要转化的文件放置于InPut中；
	此工具会将转化后文件置OutPut内
	运行ConverterToHans.py[繁体 转化到简体]
	    ConverterToHant.py[简体 转换到繁体]
        运行python  ConverterToHant.py即可
        
	注：python需是3.0以上。
